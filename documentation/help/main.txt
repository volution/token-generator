

---- How to invoke the tools

>> z-tokens patterns --help
>> z-tokens generate --help

>> z-tokens hash --help

>> z-tokens exchange keys --help
>> z-tokens exchange encrypt --help
>> z-tokens exchange decrypt --help
>> z-tokens exchange password --help
>> z-tokens exchange armor --help
>> z-tokens exchange dearmor --help
>> z-tokens exchange ssh keys --help
>> z-tokens exchange ssh wrap --help

>> z-tokens --version


---- How to generate passwords

(for quick copy-paste)
>> z-tokens g
>> z-tokens g cv-4
>> z-tokens g r-16
>> z-tokens g b-16
>> z-tokens g uuid-v4
>> z-tokens g ...

(for display and manual transcribing)
>> z-tokens generate
>> z-tokens generate -p cv-4
>> z-tokens generate ...


---- How to get help

>> z-tokens --help
>> z-tokens --help-text

>> z-tokens --readme
>> z-tokens --readme-text
>> z-tokens --readme-html
