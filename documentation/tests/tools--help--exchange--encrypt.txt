Usage:
  z-tokens exchange encrypt [OPTIONS]


Optional arguments:
  -h,--help             Show this help message and exit
  -s,--sender {sender}  (sender private key)
  -r,--recipient {recipient}

  -x,--secret {secret}  (shared secret, for additional security)
  -p,--pin {pin}        (shared PIN, for **WEAK** additional security)
  --ssh-wrap {key}      (shared SSH agent key handle)
  -M                    (treat empty arguments as unspecified)
  --empty-is-missing {bool}
