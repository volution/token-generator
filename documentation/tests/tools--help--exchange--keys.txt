>>  z-tokens exchange keys --help

Usage:
  z-tokens exchange keys [OPTIONS]


Optional arguments:
  -h,--help             Show this help message and exit
  -s                    (generate sender key pair)
  --sender {enabled}    
  -r                    (generate recipient key pair)
  --recipient {enabled} 
  -x                    (generate shared secret)
  --secret {enabled}    
  -p                    (generate shared PIN)
  --pin {enabled}       
  -o                    (**CAUTION**, generate one key, and encode it both for
                        sending and receiving)
  --self {enabled}      
  -c                    (output comments)
  --comments {enabled}
