>>  z-tokens exchange keys --help

Usage:
  z-tokens exchange keys [OPTIONS]


Optional arguments:
  -h,--help             Show this help message and exit
  -s                    (generate sender key pair)
  --sender {enabled}    
  -r                    (generate recipient key pair)
  --recipient {enabled} 
  -x                    (generate shared secret)
  --secret {enabled}    
  -e                    (generate shared seed)
  --seed {enabled}      
  -b                    (generate shared ballast)
  --ballast {enabled}   
  -p                    (generate shared PIN)
  --pin {enabled}       
  -o                    generate one key, and encode it both for sending and
                        receiving) (!!! CAUTION !!!)
  --self {enabled}      
  -c                    (output comments)
  --comments {enabled}
